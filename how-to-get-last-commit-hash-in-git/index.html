    How to Get Last Commit Hash in Git

    By Osvaldo Toja March 12, 2014

    The bash way

    1$ git log -n1 | grep commit | awk '{print $2}'
2fc77768a4f7c460be765012c9a04e9645e4520d2

    The git way

    1# short - using h
2$ git log --pretty=format:'%h' -n 1
3fc77768
4# long - using H
5$ git log --pretty=format:'%H' -n 1
6fc77768a4f7c460be765012c9a04e9645e4520d2

    You can specify the number of digits of the hash using --abbrev=n

    1$ git show --pretty=%h --abbrev=18
2fc77768a4f7c460be7
