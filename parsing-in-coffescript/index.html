    Parsing in Coffescript

    By Osvaldo Toja February 26, 2014

    Parsing with regex is simple in coffescript. Just use the same syntax as Javascript as shown below.

    More info here.

    tl;dr

    for your copy/paste please, just type the following on a coffee</console>.

    1string="wp-theme-acme-2.1.33.zip"
2pattern = /^([a-z\-]*)-([\d\.]*).zip/
3string.match(pattern)
4[component,version] = string.match(pattern)[1...3]
5component
6version

    Output

     1coffee> string="wp-theme-acme-2.1.33.zip"
 2'wp-theme-acme-2.1.33.zip'
 3coffee> pattern = /^([a-z\-]*)-([\d\.]*).zip/
 4/^([a-z\-]*)-([\d\.]*).zip/
 5coffee> string.match(pattern)
 6[ 'wp-theme-acme-2.1.33.zip',
 7  'wp-theme-acme',
 8  '2.1.33',
 9  index: 0,
10  input: 'wp-theme-acme-2.1.33.zip' ]
11coffee>

    We can store the parsed results directly into variables using something like the following.

    [component,version] = string.match(pattern)[1...3]

    Output

    1coffee> [component,version] = string.match(pattern)[1...3]
2[ 'wp-theme-acme', '2.1.33' ]
3coffee> component
4'wp-theme-acme'
5coffee> version
6'2.1.33'
7coffee>
